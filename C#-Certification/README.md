## About the Certification

The **Foundational C# with Microsoft** certification is a free professional certification created through a collaboration between Microsoft and freeCodeCamp. It includes approximately 35 hours of Microsoft Learn training content followed by a certification examination. :contentReference[oaicite:3]{index=3}

This certification validates foundational programming knowledge in C# and demonstrates practical understanding of core software development concepts.
