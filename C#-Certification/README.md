## Overview

This repository documents my successful completion of the **Foundational C# with Microsoft Developer Certification**, a collaborative certification program developed by freeCodeCamp and Microsoft.

The certification focuses on foundational C# programming concepts, problem-solving, debugging, methods, control structures, data types, and practical .NET development fundamentals. The program includes Microsoft Learn modules and an 80-question certification examination hosted by freeCodeCamp. :contentReference[oaicite:0]{index=0}

## About the Certification

The **Foundational C# with Microsoft** certification is a free professional certification created through a collaboration between Microsoft and freeCodeCamp. It includes approximately 35 hours of Microsoft Learn training content followed by a certification examination. :contentReference[oaicite:3]{index=3}

This certification validates foundational programming knowledge in C# and demonstrates practical understanding of core software development concepts.
