# 🚀 JavaScript Certification Projects

A collection of JavaScript projects built while completing JavaScript certification labs and challenges.  
This repository contains beginner-to-intermediate projects focused on DOM manipulation, APIs, OOP, ES6 concepts, data structures, and interactive web applications.

# 📚 Projects Included

## 1️⃣ Weather App

A responsive weather application that fetches real-time weather data using the FreeCodeCamp Weather API.

### Features
- Fetch live weather data
- Display temperature, humidity, wind speed, and weather icons
- Error handling
- Responsive UI

### Technologies
- HTML
- CSS
- JavaScript
- Fetch API

---

## 2️⃣ Drum Machine

An interactive drum machine that plays sounds using keyboard keys or mouse clicks.

### Features
- Keyboard and click support
- Dynamic sound playback
- Active button animations
- Display current sound name

### Technologies
- HTML
- CSS
- JavaScript

---

## 3️⃣ Markdown to HTML Converter

A markdown parser that converts markdown syntax into HTML elements.

### Features
- Heading conversion
- Bold and italic formatting
- Image support
- Link support
- Quote conversion
- Live HTML preview

### Technologies
- HTML
- CSS
- JavaScript
- Regular Expressions

---

## 4️⃣ Voting System

A voting system implementation using `Map` and `Set` to manage poll options and prevent duplicate voting.

### Features
- Add voting options
- Prevent duplicate votes
- Display poll results
- Uses modern JavaScript data structures

### Technologies
- JavaScript ES6
- Map
- Set

---

## 5️⃣ Bank Account Management Program

A simple banking system built using JavaScript classes and object-oriented programming concepts.

### Features
- Deposit money
- Withdraw money
- Transaction history
- Balance checking
- Deposit and withdrawal tracking

### Technologies
- JavaScript Classes
- OOP Concepts
